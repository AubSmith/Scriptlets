Esc - command mode
i, a - insert mode
o - open new line
:wq - write + quit
:q! - quit without saving

gg - go to first line of file
G - go to last line of file
dd - delete line
yy - copy line
p - paste line
v - visual mode
u - undo
Ctrl-r - redo
/text - search text
^ - start of current line
$ - end of current line
:%s/old/new/g - global substitute
